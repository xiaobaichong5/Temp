## 使用方法
1. Download ZIP, 解压
3. 上传至usr/plugins`目录
4. 登录后台启用即可
5. 此插件是基于原作犬の窝进行修改，由QQ爹移植Typecho，Zero适配HandSome

## 使用须知
插件需要jQuery和Font Awesome的支持
HandSome无需启用Font Awesome
